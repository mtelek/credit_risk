---------------------------------------------------------------------
✅Phase 1: Setup
---------------------------------------------------------------------

✅ Install PostgreSQL locally and set up a database called credit_risk
✅Download the Lending Club dataset from Kaggle (search "Lending Club Loan Data")
✅Explore the raw CSV — it has ~150 columns, understand what each key column means
✅Load the data into PostgreSQL using Python (psycopg2 or SQLAlchemy + pandas)
✅Define target variable: loan_status — map "Charged Off" / "Default" to 1, "Fully Paid" to 0, drop everything else (current loans, grace period etc.)

---------------------------------------------------------------------
✅Phase 2: SQL Cleaning & Feature Engineering
---------------------------------------------------------------------

Write SQL queries to:
✅Drop columns with excessive missing values (>70% null)
✅Clean text columns — emp_length is stored as "10+ years", convert to numeric
✅Compute derived features:

credit_utilization (revolving balance / revolving limit)
months_since_earliest_credit_line

✅Segment and explore default rates by loan grade, purpose, home ownership

---------------------------------------------------------------------
Phase 3: Python Modeling
---------------------------------------------------------------------

✅Pull accepted_loans table into pandas via SQLAlchemy
❌Train/test split — split chronologically by issue date, not randomly. Earlier loans train, later loans test.
❌WoE encoding — encode categorical variables using Weight of Evidence (use the scorecardpy library)

❌Logistic Regression first:
❌Fit with sklearn
❌Check coefficients make business sense (higher debt-to-income → higher default probability?)
❌Evaluate: ROC-AUC, Gini, KS statistic

❌XGBoost second:
❌Fit, tune hyperparameters with cross-validation
❌Compare performance vs logistic regression
❌Plot feature importance

---------------------------------------------------------------------
Phase 4: Scorecard & Evaluation
---------------------------------------------------------------------

❌Convert logistic regression into an actual scorecard using the scorecardpy library — this produces a points table like a real bank scorecard
❌Plot the ROC curve comparing logistic regression vs XGBoost
❌Compute and plot the KS statistic chart (shows separation between default and non-default score distributions)
❌Calibration plot — does the model's predicted probability match actual default rates?
❌Write a short interpretation: what are the top 5 most predictive features and why does that make business sense?
